Cara memakai program kami

via jupyter-notebook:
	1. download file .ipynb yang sudah kami sediakan di file
	2. buka "jupyter-notebook" yang tersedia dan upload file .ipynb yang telah di download
	3. setelah terbuka, tekan tombol play yang ada di bagian atas
	4. jika ingin run secara keseluruhan bisa ke bagian run --> run all cells
	5. selesai program bisa dijalankan
